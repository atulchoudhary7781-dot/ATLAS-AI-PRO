🚀 API USAGE EXAMPLES
════════════════════════════════════════════════════════════════════════════

# Analyze video
curl -X POST http://localhost:8000/api/video/analyze \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com/video.mp4"}'

# Predict stocks
curl -X POST http://localhost:8000/api/market/predict \
  -H "Content-Type: application/json" \
  -d '{"symbols":["AAPL","MSFT"],"timeframe":"1d"}'

# Scrape website
curl -X POST http://localhost:8000/api/scraper/analyze \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","depth":2}'

# Audit code
curl -X POST http://localhost:8000/api/audit/code \
  -H "Content-Type: application/json" \
  -d '{"code":"print(\"hello\")","language":"python"}'

❓ TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Backend won't start?
  → Check if port 8000 is free
  → Verify Python 3.11+
  → Check API key is set

Frontend won't load?
  → Check backend is running first
  → Backend must be on localhost:8000
  → Clear browser cache

CLI says "cannot connect"?
  → Backend not running
  → Port 8000 not accessible
  → Check firewall settings
